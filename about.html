About this Web App

Why does this app exist?

A lot of people say they find it hard to edit the wiki. This is not a new insight, others have realised it many years ago, for example seamonkey420, who wrote a wiki page creator as a batch script in 2017 to help with that and make the wiki more accessible.

In order to make it cross platform and more accessible, I took the opportunity to learn web development and chose this as my first project.

Who made and maintains this?

The page creator was made by me, Lenni, from the Galactic Hub Eissentam Scribes. I make sure the data is correct after each NMS update, and also fix any bugs that you report.

I'm also the director of the Galactic Hub Eissentam and a wiki admin.

How can I contribute?

First of all, thanks for wanting to help!

There are multiple ways to contribute. Either you directly add to the codebase, or you hunt bugs and report them. But even just using the page creator and creating pages with it gives me feedback about potential pain points when I review the created pages.

To contribute, you need to fork the repo. You can then edit the code and open a pull request back to the main repository when you're done.

If you come across a bug and want to report it, use either the feedback form, text me on Reddit or Discord, or submit a GitHub issue.

Can I make my own version?

Yes, absolutely!

The app is licensed under the GPLv3 license, meaning you can make your own version, as long as it stays open source and also has the GPLv3 license.

Do you collect data?

No, the app collects no data. The only data that is saved to your local drive is your currently selected theme. Everything you do here stays on your local computer.

What are the plans for the future?

Once the Hub specific page creator is done, the project will be expanded to a more general page creator, so that everyone can use it. It will also be localised in some form, meaning other language wikis can be supported, such as German.
